# Coral E-commerce
This is a frontend ecommerce project developed using HTML, CSS and Javascript.  

The application fetches product data from the Fake Store API and displays them dynamically in a product catalog.

Users can browse products, filter them by category and add items to a shopping cart. The project focuses on practicing API consumption, DOM manipulation and building interactive frontend features.

## Technologies Used

- Sass
- Gulp
- Javascript

## Demo
https://coral-store-project.netlify.app/

## Instalation

### Install dependencies
````bash
npm install
````

### Run development server
```bash
npm run dev
```
